## Development Details

This is a Unity-based Quiz game project that includes runtime gameplay, editor tooling, and UI built using Unity **UI Toolkit**.

The runtime system handles the quiz flow, including question display, answer selection, and scoring. The UI is implemented using UI Toolkit, with structured UXML and styling to manage the quiz interface.

In addition, a custom editor tool was developed to manage quiz questions directly inside the Unity Editor. This allows quiz content to be created and edited without modifying code.

The project separates data, editor tooling, gameplay logic, and UI layers to keep the system organized and maintainable.

## My Tasks

* Implemented the core quiz gameplay loop (question → answer → result).
* Built Runtime UI and Editor tool using Unity UI Toolkit.
* Created a Unity Editor tool for managing and organizing quiz questions.
* Structured quiz data to separate content from gameplay logic.
* Connected editor-managed data with the runtime quiz system.

## Result

The project provides a functional quiz system with an editor workflow and UI built using Unity UI Toolkit.

It allows:

* Playing a multiple-choice quiz game in Unity
* Managing quiz questions through a custom editor tool
* Displaying quiz UI using UI Toolkit
* Tracking and displaying player scores
* Separating gameplay logic, UI, and quiz data 

The project demonstrates basic gameplay programming, Unity Editor tooling, and UI Toolkit implementation in a single project.
