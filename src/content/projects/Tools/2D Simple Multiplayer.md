## Development Details

This is a 2D side-scroller multiplayer/co-op game built in Unity. The game features light competitive gameplay where players complete tasks while also being able to interfere with each other.

The project uses **Unity Netcode for GameObjects** for multiplayer synchronization, along with **Unity Lobby and Relay services** to enable online multiplayer sessions without requiring direct peer-to-peer connections.

Core gameplay focuses on player movement, task interaction, and simple competitive elements within a shared multiplayer environment.

## Result

The project is a functional 2D multiplayer prototype featuring shared gameplay where players can complete tasks and interact within the same environment.

It demonstrates:

* Unity Netcode-based multiplayer implementation
* Online matchmaking using Lobby and Relay services
* Basic competitive/co-op gameplay mechanics
* Networked player movement and interaction synchronization
