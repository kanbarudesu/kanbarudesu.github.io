## Overview
Slime Idle Simulation is a simple idle simulation game where players take care of slimes by feeding them and expanding their slime population. Players can buy more slimes, purchase food, and upgrade food spawn timers to support growth. Each slime generates resources over time, but it will die if its hunger drops to zero, creating a balance between expansion and survival. The main focus of this project was to implement a simple state machine system to control slime behavior such as moving, growing, and searching for food.
