## Development Details

Outreach is a short incremental game created for the [**GameDev.tv Game Jam 2026**](https://itch.io/jam/gamedevtv-jam-2026). The game is built around building and expanding a network of relay towers across a fog-covered map to reach a distant beacon.

The core design focuses on progression through resource gathering, unlocking upgrades, and gradually automating systems to expand the player’s reach across the map.

---

## My Tasks

* Implemented [skill tree system](https://github.com/kanbarudesu/SkillTree-Graph) for unlocking buildings and units.

## Result

The project was submitted to the **GameDev.tv Game Jam 2026**, where it ranked **#31 out of 833 entries overall**.

The game demonstrates:

* Incremental gameplay loop design (harvest → research → automate → expand)
* Skill tree-based progression system
* Simple automation and network-building mechanics
* Rapid development of a complete playable game within 10 days of gamejam timeframe

---

## Gameplay Loop

* **Harvest**: Manually gather resources to start progression
* **Research**: Spend resources in a skill tree to unlock buildings and units
* **Automate**: Build structures that automate resource generation
* **Connect**: Expand a network of towers to reach the beacon

**Controls**

* Left Click: Harvest resources / interact with UI
* Right Click: Pan camera
