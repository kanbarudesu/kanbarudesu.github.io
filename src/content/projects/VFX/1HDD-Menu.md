## Overview
A recreation of the Hollow Deep Dive menu VFX from Zenless Zone Zero, focused on matching the original shape and timing of the effect.

I built this in Unity using Shader Graph and VFX Graph. I created a custom shader to form the hollow spherical zone, while using the VFX Graph system to generate a blocky volumetric effect that responds to the sphere’s position and size, matching the in-game VFX. I also created a fog shader to add depth and atmosphere.
