## Overview
A vertical pillar beam VFX I created, based on a tutorial by [Gabriel Aguiar Prod](https://www.youtube.com/watch?v=eYzihHPtGJE).

Instead of following the original VFX Graph approach, I remake the effect using Unity’s Particle System. I started by creating a custom cylinder mesh as the base shape for the beam, then created several custom shaders to define the core beam look. I combined this with additional supporting particle systems to add motion and detail, and assembled everything into a vertical pillar beam effect.
